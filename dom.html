My Class Notes:

getElementById();

The document method getElementById() will go into your document and look for that specific element

getElementByTagName();

The document method getElementByTagName() will give you all the elements with a particular tag name in a collection.

querySelector();

The document method querySelector(); will return the first match of a query

The document method querySelectorAll(); will get all the elements that match a certain selector. It returns a nodeList of elements that you have to loop through.

Working with Properties

.style.property = "value"; can be used to change the color of an element.

.innerHTML = "value"; can be used to change the html inside any element on a web page.

.className = "value"; allows you to set the class or classes for an element.

Working with Element Methods

setAttribute(); let's you set any type of attribute on an element
